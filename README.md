# Retro Pong

A retro-styled pong game built using HTML, CSS, and JavaScript.

## Features

- Smooth paddle controls (keyboard + touch)
- Level progression every 5 hits
- Football emoji with trail effect
- Retro theme with pixel font
- Dynamic background colour changes
- Game Over screen with score and best score

## To Run

Just open `index.html` in your browser or host with GitHub Pages.
